## Overview üìù

digitalCV is a simple and solid personal portfolio website that showcases your skills, projects, and achievements. It is fully functional and contains a HTML route to a nice meditation how-to guide. Whether you are looking for a job, a client, or a collaborator, digitalCV can help you make a great first impression online. üñ• üìÇ
